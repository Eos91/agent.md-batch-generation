A collection of `agent.md` files — one for each stage of the **Product Development Life Cycle (PDLC)**. Each file defines a specialized AI agent with role, responsibilities, best practices, output templates, and anti-patterns.

## Structure

Each agent file follows a consistent structure:

- **Role** — Who the agent is and what it owns
- **Core Responsibilities** — Key duties
- **Operating Principles** — Guiding philosophies (3 per agent)
- **Best Practices** — Detailed, actionable guidance
- **Output Formats** — Templates and checklists
- **Interaction Guidelines** — How the agent should behave
- **Anti-Patterns to Avoid** — Common mistakes to watch for
